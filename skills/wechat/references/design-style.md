# WeChat Official Account Visual Design Guide

## 封面风格分类

### 1. 简约专业型 — Best for: 科技、财经、企业号
- 干净背景 + 大标题
- 品牌色 + 白色/深色
- 无多余装饰
- 适合每期统一模板

### 2. 图文并茂型 — Best for: 生活、美食、旅行
- 摄影图片为主体
- 文字叠加在图片上（需要遮罩层保证可读性）
- 温暖色调
- 每期可以不同但色调统一

### 3. 插画风格型 — Best for: 教育、亲子、文化
- 手绘或矢量插画
- 活泼但不幼稚
- 明快配色
- 适合系列化设计

### 4. 数据驱动型 — Best for: 行业报告、深度分析
- 图表/数据可视化为主
- 专业感强
- 蓝灰色调
- 突出关键数据

### 5. 品牌强化型 — Best for: 品牌公众号、公司号
- 强品牌视觉识别
- 固定版式 + 品牌色
- Logo 常驻
- 系列感最强

## 配色方案

### 科技 / 互联网
- Primary: 深蓝 `#1A237E`, 科技蓝 `#2196F3`
- Accent: 青色 `#00BCD4`
- Background: 深色 `#0F1419` 或纯白

### 财经 / 商业
- Primary: 藏蓝 `#0D2137`, 金色 `#C9A44A`
- Accent: 红 `#C62828` (涨), 绿 `#2E7D32` (跌)
- Background: 白色或浅灰

### 生活 / 美食
- Primary: 暖棕 `#8B6F47`, 奶油 `#FBF3EB`
- Accent: 橄榄 `#6B7B3A`, 珊瑚 `#FF7043`
- Background: 暖白 `#FAF7F2`

### 教育 / 知识
- Primary: 学院蓝 `#1565C0`, 暖黄 `#FFD600`
- Accent: 绿 `#4CAF50`
- Background: 浅蓝 `#E3F2FD`

### 健康 / 医疗
- Primary: 青绿 `#00897B`, 白
- Accent: 浅珊瑚 `#FFAB91`
- Background: 薄荷 `#E0F2F1`

## 头条封面 (900x383) 排版规则

### 字体规范
- 标题字号: 36-52px (相对于 900px 宽度)
- 最多 2 行标题，每行不超过 10 个汉字
- 字重: 粗体或超粗体
- 字体推荐: 思源黑体 (Noto Sans SC), PingFang SC, 阿里巴巴普惠体

### 安全区域
```
┌──────────────────────────────────┐
│  ┌──── 安全区域 (内缩 40px) ────┐│
│  │                              ││
│  │   标题区域                   ││
│  │   (居中或左对齐)             ││
│  │                              ││
│  │        [视觉元素]            ││
│  │                              ││
│  └──────────────────────────────┘│
│  ← 40px →                ← 40px →│
└──────────────────────────────────┘
```

### 布局模式

**居中标题型:**
```
┌──────────────────────────────────┐
│          [背景/渐变]             │
│                                  │
│        大 标 题 文 字            │
│        副标题或日期              │
│                                  │
└──────────────────────────────────┘
```

**左文右图型:**
```
┌──────────────────────────────────┐
│                    │             │
│  标题文字          │  [图片/     │
│  副标题            │   插画]     │
│                    │             │
└──────────────────────────────────┘
```

## 次条缩略图 (200x200) 设计

200x200 极小尺寸，设计策略完全不同：

```
┌──────────┐
│          │
│  [图标]  │   ← 单一图标，无文字
│          │
└──────────┘
```

- **纯图标**: 一个简洁的图标/符号
- **纯色块**: 品牌色背景，可选简单图形
- **数字**: 如果是系列，可用大数字（"03"）
- **绝不放文字**: 200px 上文字根本看不清

## 深色模式适配

微信支持深色模式，封面需要注意：
- 避免纯白 `#FFFFFF` 背景 — 深色模式下非常刺眼
- 使用 off-white `#F5F0E8` 或更深的背景色
- 确保在深色背景上文字仍然可读
- 测试：深色模式下封面边缘是否与 UI 背景融为一体

## 设计反模式

1. **封面文字 = 标题**: 用户已经看到标题了，封面不要重复
2. **文字太小**: 手机上封面会被缩小，字必须够大
3. **元素太多**: 900x383 是矮宽比例，空间有限，最多 3 个视觉元素
4. **忽略次条**: 次条缩略图也需要设计，不能随便放张图
5. **不统一**: 每期风格差异太大，削弱品牌认知
6. **纯白背景**: 深色模式下效果差
